# Food Order DevOps Project

A full-stack food ordering application with a complete DevOps pipeline.

## Description
Users can select a food item (Pizza, Sandwich, Pasta, Chicken) and place an order with their phone number.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Containerization: Docker
- Orchestration: Kubernetes (k3s)
- CI/CD: GitHub Actions
- GitOps: ArgoCD
- Security: Trivy
- Monitoring: Prometheus + Grafana
- Agile: GitHub Projects

## How to Run
docker compose up --build
